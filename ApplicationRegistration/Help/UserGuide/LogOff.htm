LogOff

 

You can log off from Registration and Activation Server using the LogOff link available at the right-top portion of the server.

 

The logOff link is visible in all screens of the server.

 

Notes:

  1. To conserve the server resources, Registration and Activation Server automatically logs you out if you do not use the application for a specified period of time. This is normally 15 minutes and may vary depending on the setup done.

  2. If you have been logged out, you can start using the software by simply logging in again.
