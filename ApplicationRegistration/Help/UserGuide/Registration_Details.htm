Registration Details

 

Target Users: Portal Admin

  1. Lists the registration details.

  2. Using navigation tool buttons like    Portal Admin can do CURD [Create Update Read Delete] operations.

  3. Page navigation is provided to display more records in a single page.

  4. The Delete operation will move the record to a separate table in the database. This is to track and allow re-registration from the same machine.
