Add a new Product Definition

 

To Add a new product definition, click on button on the Products screen. This displays the Add product definition screen shown in Figure 4.1.

 

To Add a new product definition:

  1. Fill in the following fields, displayed in the "Product Information" box:

  • Product Name - Enter the name of the product. Example: Sharpdesk 3.3

  • Product ID - Enter a unique identifier for the product. For example, SD3.3 for the product Sharpdesk 3.3 (Note: The same product with different version number is considered as distinct product)

  • Version - Enter the Version number of the product. For example, Enter 3.3 for Sharpdesk.

  • Build Number - Enter the build number of the product.

  • Family - Enter the category of the product, to which it belongs to. For example, Sharpdesk belongs to "Document Management" family.

  • Company - Select the company name of the product from the drop-down list.

  • Logo - Click on "Browse" button, to select the location for the logo of the product.

  • Allow Registration - Check this box, to allow registration for the product from the client machines.

  1. After entering the above information, click on Add button. It will add the product to the database and displays the updated Products screen. And, also shows the message "Product added successfully" for the newly added product. See the Figure 4.2.

  2. When the product definition is created, Registration Web Application generates “Registration AccessID” and “Registration Access Password”. You can see these details on the Products screen for the newly created product.

Notes:

  1. The fields marked with * are mandatory.

  2. To edit or delete a product definition, select the product, and use the buttons.

  3. Use the button, to export the details of a selected product to an Excel document.
