Other Configurations

 

Target Users: Portal Admin

 

Portal Admin can configure the application settings like:

 

  1. Maximum records in reports for the option 'On Screen'.

  2. Maximum records in reports for the option 'CSV'.

Options 1 and 2 are basically to avoid the processing the large result set, that Report can generate. With this settings, only the max number specified records will be displayed on Screen/written to CSV files.

  1. SMTP Server.

Admin can configure the SMTP server name. This server will be used to send the all the emails from Registration Server.
