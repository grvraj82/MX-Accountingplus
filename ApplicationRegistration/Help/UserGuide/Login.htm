Registration and Activation Server is a License Management System for the SHARP products registered in the server. And, it also allows the clients to activate their SHARP products.  

 

LogOn to Server

To start using Registration and Activation Server, you need to access the software through a web browser.

Open your browser and type in the URL (http://sharpdesk/ApplicationRegistration_sprint11) to access Registration and Activation Server. This will open up the page from where you can logOn to the server.

After entering the Server URL into the browser, the logOn page will be displayed. Enter your user name and password and click on the LogOn button.

If you have entered a valid User Name and password, you will be redirected to a different page, depending on the role (Portal Admin or Product Admin or Product Customer Representative) defined for you.
