Managing Registration details

 

Target Users: Portal Admin

  1. Portal Admin can Add or Update Registration details.

  2. Adding registration feature helps in registration of following types:

  • Phone

  • Fax

  • Email

  1. Serial Key validation takes place, while adding registration record.

  2. After adding registration record, Email will be sent to the user.

  3. The fields in “Other” category are custom fields.

  4. Any number of Custom fields can be configured.

  5. Custom fields in other sections are generated dynamically as form elements.
