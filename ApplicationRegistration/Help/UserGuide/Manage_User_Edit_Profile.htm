Manage User/Edit Profile

 

Target Users: Portal Admin/Logged-in user

  1. Portal Admin can Add/Edit any user’s information. But, other users (Product Admin/Product CR) can only edit their own user information.

  2. User can select the preferred product. When the user logs in, by default the preferred product will be set as the selected product.

  3. Select the check box-“Access Enabled” to give access permissions to the User for the Product Registration and Activation Web application.
