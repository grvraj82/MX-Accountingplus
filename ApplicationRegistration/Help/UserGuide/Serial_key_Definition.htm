Serial key Definition

 

Target Users: Portal Admin

  1. Lists the registered Serial Keys.

  2. Provides page navigation, to support displaying millions of records.

  3. Using navigation tool buttons like Portal Admin can do CURD [Create Update Read Delete] operations.

  4. Particular Serial Key can be deleted only when there are no registrations exists for that Serial Key.

  5. Displays the records in different color, where all the allowed number of licenses are in use.

This screen provides navigation to see all the registrations for a selected serial key [by clicking on “Registered” column data, User can navigate to registered records].
