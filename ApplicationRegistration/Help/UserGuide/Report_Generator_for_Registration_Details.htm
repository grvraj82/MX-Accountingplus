Report Generator for Registration Details

 

Target Users: Portal Admin/Product Admin

  1. Portal Admin/Product Admin can generate a report for a registered product. You can select the product from the Product Drop-Down list shown in the right-most top corner on the Reports screen.

  2. Report Generation:

  • Date Format:

To generate a report based on the registration dates, select “Registration Date” radio button and specify the dates in the “From” and “To” date fields. You can specify the dates manually or using the Calendar control.

For example, the screen in the Figure 4.1 shows the report generated based on the registration dates for the Sharpdesk 3.3 product.

  • Data Column:

        Select the Data Column radio button, to generate a report based on:

      1. User Information like First Name, Last Name, Serial Key, or Activation Key etc...

      2. Purchase Information like Sharp Dealer Name or Sharp MFP Model.

      3. Company Information like Company Name or Organization Type etc…

      4. System Information like Computer Name, IP Address, or Operating System etc…

For example, the screen in the Figure 4.2 shows the report generated based on Activation code for the Sharpdesk 3.3 product.

  1. Report Format:

  • On Screen: This option displays the Query results on the screen immediately. Since there is a limitation on the data that can be displayed/Rendered to web browser, the record limit can be set to “Configuration >> Others”

  • CSV File: This option helps in writing the Query results to CSV file. Since the query can result in millions of records, and cannot be written to CSV Files, the record limit can be set to “Configuration >> Others”.

  1. First column (i.e. Computer Code) of the Resulted data is linked to editing Registration screen.

  2. Using icon, user can print the generated Report.
