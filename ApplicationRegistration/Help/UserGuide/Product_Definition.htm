Products

 

The Products screen is the first screen that you will see, when you first logged on to the server. This screen displays the list of products that are defined in the server. And, only for these defined products, clients can register and activate their SHARP products before the trial period expires.

 

You can do the following operations on this Products screen:

  • Add - Using button, you can create a new product definition.

  • Edit - Using button, you can edit a product definition.  

  • Delete - Using button, you can delete a product definition.

  • Export to - Using button, you can export the selected product details to an Excel document.

The fields in the Products screen are explained below:

  • Product Name - This column displays the product names defined in the server.

  • ID - The unique ID of the product will be displayed.

  • Logo - The product logo will be displayed. For example: Sharpdesk product logo will be displayed for the Sharpdesk product under logo column.

  • Version - The version of the product will be displayed.

  • Build - The product build number will be displayed.

  • Family - The domain of the product will be displayed. Example: For Sharpdesk, “Document Management” will be displayed under this column.

  • Company - The company name of the product will be displayed.

  • Registration Access Id - The unique Login ID to access the product using Web Service API. This ID will be generated automatically, when a product is created.

  • Registration Access Password - The unique Password to access the product using Web Service API. This ID will be generated automatically, when a product is created.

  • Registration Allowed - If it is "True", it allows registration for the product from the client machines. Else, it does not allow registration for the product.
