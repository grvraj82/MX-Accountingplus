Portal Admin Users can do the following operations:

  1. Logon to the Activation Server

  2. Create Product Definition

  3. Create Users

  4. Create Roles

  5. Assign Roles for the Users created

  6. Manage Serial Keys for any product [Add/Delete/Modify]

  7. Manage Registrations for any product [Add/Delete/Modify]

  8. Generate Reports

  9. Manage Server Configurations

  10. Change Password

  11. Manage Profile
