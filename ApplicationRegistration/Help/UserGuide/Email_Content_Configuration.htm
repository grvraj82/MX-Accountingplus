Email Content Configuration

 

Target Users: Portal Admin

 

Portal Admin can manage the Activation Email content. This feature helps the admin to configure the product specific activation email content.

  1. While composing the Product Registration email, application looks for the configured content of the product.

  2. If the content is missing for a particular product, then the content, From, CC, BCC, Subject will be picked from the “All” product content.

  3. Addition to this content, the following information will be added to the email content.

  • User Name

  • Product

  • Serial Key

  • Activation Code
