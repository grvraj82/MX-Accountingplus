Managing Serial Key Definition

 

Target Users: Portal Admin

  1. If Serial Key information is available, Portal Admin can add it to server.

  2. Serial Key information [Serial Key, Total Licenses] gets added automatically, when authenticated client sends the request for registration.

This feature is just to avoid having the serial key information on the server, before allowing registrations.

 

A serial Key validation happens on Web server, before adding Serial Key information to Database.
