Role Definition

 

Target Users: Portal Admin

  1. Lists all the defined Roles for Product Registration and Activation Web Application.

  2. Using navigation tool buttons like Portal Admin can do CURD [Create Update Read Delete] operations.

  3. Using button, Portal Admin can assign the roles to the users.

  4. The access control in the application is based on the user role.
