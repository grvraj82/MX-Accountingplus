User Definition

 

Target Users: Portal Admin

  1. Lists all the users of Product Registration and Activation Web Application along with their details (Like ID, Name, Company etc…).

  2. Using navigation tool buttons like Portal Admin can do CURD [Create Update Read Delete] operations.

  3. Using button, Portal Admin can export the users List to Microsoft Excel.
