Web Application Configuration

 

Target Users: Portal Admin

 

Portal Admin can update the configuration data of Product Registration and Activation web application.

The configuration includes the following areas:

  1. Managing Lookup tables like MFP Models, Job functions, Countries, States

  2. Configuring Field Access Definition

  3. Configuring Activation Email Content

  4. Managing Support contact Details

  5. Others:

  • Maximum records in reports for the option 'On Screen'

  • Maximum records in reports for the option 'CSV'

  • SMTP Server
